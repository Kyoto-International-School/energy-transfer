# Agents.md

## Project overview

This project is a client-side-only React application that serves as an Energy Transfer Diagram Editor. It utilizes Vite, React, TypeScript, and React Flow to provide a drag-and-drop interface for creating and managing energy transfer diagrams. The application is designed to be deployed on GitHub Pages. It needs to support touchscreens and pointer devices for drag-and-drop functionality.

## High-level stack

- Vite + React + TypeScript + React Flow
- GitHub Pages for hosting
- localStorage for persistence
